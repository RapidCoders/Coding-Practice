# Q.1 Write a function that determines if a given positive integer is a power of four.
```
Input: 16
Output: true
Explanation: 16 is 4^2.

Input: 64
Output: true
Explanation: 64 is 4^3.

Input: 5
Output: false
Explanation: 5 is not a power of four.

Input: 1
Output: true
Explanation: 1 is 4^0.
```
